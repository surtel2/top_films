Empire's Top Greatest's Films

The steps which were implemented during this project are:
1. Web-scrapping 500 titles of films (via Python)
2. Creating local database with tables in Postgres.
3. Fetching the films data from open API about films (http://www.omdbapi.com/)
4. Cleaning and formatting data using SQL queries (select + dynamic query)
5. Loading data into Power BI and creating a report
6. Light-dark theme, bookmarks, tooltips, DAX measures and columns were used.
